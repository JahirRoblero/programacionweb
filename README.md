# Actividad 1 - Programación Web

# Alumno: GOMEZ ROBLERO ANGEL JAHIR

# Docente: ADELINA MARTINEZ NIETO

Actividad 1 de la materia de Programación Web, impartida por la docente Adelina Martínez Nieto.

Este repositorio contiene los ejercicios prácticos enumerados del 1 al 10, realizados en clase. Todos los ejercicios se encuentran integrados en un menú principal dentro del archivo index.html.

Desde el menú principal se puede acceder a cada ejercicio dando clic en cualquiera de las opciones disponibles.

En este repositorio se incluyen 10 ejercicios. El primer ejercicio es un “Hola mundo”, como introducción al uso de HTML. En el ejercicio número 2 se muestra un ejemplo de una página con formato de texto, usando negritas y énfasis. En el ejercicio número 3 se trabajan enlaces en HTML, mientras que en el número 4 se incorpora el uso de imágenes.

En el ejercicio número 5 se combina texto con enlaces. En el ejercicio número 6 se utilizan secciones semánticas básicas. En el ejercicio número 7 se presenta una receta de tacos al pastor. En los ejercicios 8 y 9 se implementan varios de los elementos vistos anteriormente, como tablas, formularios, imágenes, enlaces y estructura HTML.

Finalmente, en el ejercicio número 10 se incorpora JavaScript para crear una calculadora que convierte grados Celsius a Fahrenheit.

El objetivo de esta actividad es reunir los ejercicios realizados en clase dentro de un solo repositorio de GitHub y publicarlos mediante GitHub Pages, para poder visualizarlos desde un navegador web.
